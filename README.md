# sub4me.ai

sub4me.ai is a modern airtime and data purchase platform built with simplicity, speed, and accessibility in mind. The platform is designed to make buying airtime and mobile data effortless for everyone — including users who are not tech-savvy.

## ✨ Features

- Buy Airtime
- Buy Mobile Data
- Save frequently used contacts
- Add, edit, and delete saved contacts
- Quick recipient selection from contacts
- Option to save numbers during transactions
- Automatic network detection
- Manual network selection fallback
- Real-time form validation and error handling
- Push notifications
- Fully responsive design
- Smooth and interactive user experience

## 🎯 Project Goal

The goal of sub4me.ai is to create an extremely simple and intuitive platform where users can purchase airtime and data in just a few clicks with minimal friction.

The UI/UX is heavily focused on:
- Simplicity
- Accessibility
- Speed
- Ease of use
- Mobile-first interaction
